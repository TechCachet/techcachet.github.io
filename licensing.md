# Licensing: MIT repo vs. paid Marketplace app

The **Forge / Linked SLA Alerts** source code may live in a public GitHub repository that still shows an **MIT** `LICENSE` file. The same product is offered on the **Atlassian Marketplace** as a **paid** app.

**Those are different contexts.** MIT addresses rights in the **repo**; Marketplace customers are subject to **Atlassian checkout terms**, your **listing**, and Tech Cache **[Terms of Service](/terms)**.

**Do not assume** MIT means the installed app is free for enterprise use or that Marketplace obligations disappear.

The canonical technical note for contributors and maintainers is in the app repo:

**[github.com/TechCache/sla-link-inspector](https://github.com/TechCache/sla-link-inspector)** → `sla-link-inspector/docs/LICENSING-NOTE.md`
